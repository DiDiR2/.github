# DiDiR2 - Direct Drive Robot 2

No reduction, no gears, no backlash, no friction, no wear, no complexity, ... Just drive!

DiDiR2 has 5 direct drive BLDC motors:

- 2 for locomotion
- 1 for equilibrum
- 2 for arm.
